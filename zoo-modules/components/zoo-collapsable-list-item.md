## Example usage

## Attributes

| **Name**  | **Description**                                        | **Optional** |
| :-------- | :----------------------------------------------------- | :----------: |
| `opened-by-default` | element will be initially opened |   &#10003;   |
| `border-visible` | border will remain visible when the component closed |   &#10003;   |
