# [Spinner](#spinner)

Spinner svg.

***

> Spinner has `position: absolute`, you have to place it inside an element with `position: relative`.

## Example usage

```HTML
<div style="position: relative; height: 100px; width: auto">
	<zoo-spinner></zoo-spinner>
</div>
```

## Rendered

<div style="position: relative; height: 100px; width: auto">
	<zoo-spinner></zoo-spinner>
</div>

## CSS Variables

|    **Name**     | **Description**  |
| :-------------: | :--------------- |
| `--primary-mid` | color of spinner |
