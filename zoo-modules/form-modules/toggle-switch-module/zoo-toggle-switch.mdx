# Toggle Switch

Input component which provides styles for slotted `input`.

## Attributes
|      **Name**     |           **Description**                                          | **Type** | **Default** | **Optional** | 
| :---------------: | :-----------------------------------------------------------------:|:--------:|:-----------:|:-----------:|
| `labeltext`       | text to be presented as a label                                    | text     |             |   ✔
|   `infotext`      | text that should be displayed as additional info below the input   | text     |             |   ✔

## CSS Variables
|        **Name**        |   **Default**  |
| :--------------------: |:--------------:|
| `--primary-mid`        |    `#3C9700`   |
## Slots
|      **Name**    |           **Description**          | **Optional** | 
| :--------------: | :---------------------------------:|:-----------:|
| `input`          | `<input id="input-id" slot="input" type="checkbox"/>`| ✘
| `label`          | `<label for="input-id" slot="label">An example input</label>`| ✔

## Example usage:
```
<zoo-toggle-switch infotext="Additional helpful information for our users">
	<label for="input-toggle" slot="label">Toggle switch</label>
	<input id="input-toggle" slot="input" type="checkbox"/>
</zoo-toggle-switch>
```
