# Segmented Buttons

Container omponent which provides styles for box around multiple `zoo-button` components.

## Slots
|      **Name**    |           **Description**          | **Optional** | 
| :--------------: | :---------------------------------:|:-----------:|
|                  | `<zoo-button type="primary">Button 1</zoo-button>`| âœ˜

## Example usage:
```
<zoo-segmented-buttons style="width: 600px">
	<zoo-button type="primary">
		<button type="button">Button 1</button>
	</zoo-button>
	<zoo-button>
		<button type="button">Button 2</button>
	</zoo-button>
	<zoo-button>
		<button type="button">Button 3</button>
	</zoo-button>
</zoo-segmented-buttons>
```
