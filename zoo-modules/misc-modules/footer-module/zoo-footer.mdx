# Footer

Container footer component.

## Attributes
|   **Name**  |                 **Description**              | **Type** | **Default** | **Optional** |
| :---------: | :-------------------------------------------:|:--------:|:-----------:|:-----------:|
| `copyright` | arbitrary text to show in bottom left corner | text     |             |   ✔
## CSS Variables
|        **Name**        |   **Default**  |
| :--------------------: |:--------------:|
| `--primary-light`      |    `#66B100`   |
| `--primary-mid`        |    `#3C9700`   |
## Slots
|     **Name**   |           **Description**          | **Optional** | 
| :------------: | :---------------------------------:|:-----------:|
|                | arbitrary content                  | ✔

## Example usage:
```
<zoo-footer copyright="zooplus AG">
	<zoo-link href="https://github.com/zooplus/zoo-web-components" text="Github"></zoo-link>
	<zoo-link href="https://www.npmjs.com/package/@zooplus/zoo-web-components" text="NPM"></zoo-link>
</zoo-footer> 
```
