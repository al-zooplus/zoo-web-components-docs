# Spinner

Spinner svg.
Note that spinner has `position: absolute`, you have to place it inside an element with `position: relative`.
## CSS Variables
|        **Name**        |   **Default**  |
| :--------------------: |:--------------:|
| `--primary-mid`        |    `#3C9700`   |

## Example usage:
```
<zoo-spinner></zoo-spinner>
```
